Edit API Status

To edit the scorecard, you will need to get a GitHub OAuth token.

Getting a GitHub token:

  1. Go to GitHub's Personal access tokens in settings.
  2. Click Generate new token
  3. Enter a token description
  4. Select the Repo checkbox
  5. Click Generate
  6. Select Generate

Get started at GitHub's Personal access tokens page.

You can use the following forms to edit the API's status and various other data points that appear on the main page.

Please note:

  • Changes may take up to five minutes to display on the dashboard as the site must rebuild after saving.
  • Adding and deleting sites from this page may cause the list of sites to temporarily change. Refresh the page to reload them.
  • Changing the name of an API will cause a new API to be saved. Delete the old version after changing the name of an API.
