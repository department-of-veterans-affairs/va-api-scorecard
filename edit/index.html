Edit APIs

Once you have a valid Github OAuth token entered into the text box below, the page will allow you to edit the api's. Without a valid API token, your changes will not be saved.

How to get a token

  1. Go to Github Personal Auth Token
  2. Click 'Generate new token'
  3. Enter a token description
  4. Select 'Repo' checkboxes
  5. Select 'Generate'
