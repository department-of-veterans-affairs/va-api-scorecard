Edit APIs

To edit the scorecard, you will need to get a GitHub OAuth token.

Getting a GitHub token:

  1. 1. Go to GitHub's Personal access tokens in settings.
  2. 2. Click Generate new token
  3. 3. Enter a token description
  4. 4. Select the Repo checkbox
  5. 5. Click Generate
  6. Select Generate

Get started at GitHub's Personal access tokens page.
